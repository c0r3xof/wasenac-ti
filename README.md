Projeto: WA Senac Chat

Descrição:
Sistema de chat interno desenvolvido para comunicação entre alunos dentro da sala de aula no SENAC.

Tecnologias utilizadas:
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Next.js

Funcionalidades:
- envio de mensagens em tempo real
- identificação de usuário
- chat em grupo

Objetivo:
Projeto desenvolvido para estudo, conversas, dúvida com outros alunos do senac.
